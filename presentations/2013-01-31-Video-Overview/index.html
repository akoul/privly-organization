Changing the rules: Privly

Privly is a web privacy stack implementing a design pattern

web Privacy stack: Injected Application + Host Page + Extension + Cryptography Library + Content Server

Design Pattern: iframe Injection

System Architecture

Why so...complicated?

  • This is platform indifferent and works with most APIs
  • Bridging unsafe JavaScript with a private key is challenging
  • Allows for a separation of concerns, and the development of a large number of cryptographic operations
  • Allows for "branding" of each cryptographic operation (people are terrible at security)
